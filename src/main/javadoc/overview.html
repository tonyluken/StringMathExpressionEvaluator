Getting Started

The code snippet below shows a simple example of using StringMathExpressionEvaluator to find the value of three expressions. 


    StringMathExpressionEvaluator smee = new StringMathExpressionEvaluator();

    String[] expressions = {"2*(3+7)", "3.2*sin(0.56)", "atan2(-1, -1)"};
    double value;
	
    for (expression : expressions) {
        try {
            value = smee.evaluate(expression);
            System.out.println("\"" + expression + "\" evaluates to " + value);
        }
        catch (InvalidMathExpressionException ex) {
            System.out.println("\"" + expression + "\" failed to evaluate due to: " + ex.getMessage());
        }
    }

Executing this code results in the following output:


    "2*(3+7)" evaluates to 20.0
    "3.2*sin(0.56)" evaluates to 1.699795833346827
    "atan2(-1, -1)" evaluates to -2.356194490192345

See the StringMathExpressionEvaluator Wiki for a complete list of operators and functions that can be used in expressions.

By default, trig functions expect their input arguments to be in radians and the inverse trig functions return their results in radians. This can be changed by calling the setDegreeMode() method.
